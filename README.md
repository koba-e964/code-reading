# このリポジトリーのポリシー
- 外部から頂いた PR はマージしない。
- 誤字脱字などを見つけていただいた際に PR などを頂くのは歓迎するが、その変更を自分でコミットしてその PR はマージせずに閉じる。
- 何かを書いてほしいという要望は受け付けない。
- 誤字脱字の指摘は PR か各種 SNS のアカウントまで。
